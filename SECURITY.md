# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | Yes                |

## Reporting a Vulnerability

If you discover a security vulnerability in tailwind_flutter, please report it responsibly.

**Do not open a public issue.**

Instead, email **nick@nickdemari.com** with:

1. A description of the vulnerability
2. Steps to reproduce
3. Potential impact
4. Suggested fix (if any)

You should receive a response within 48 hours. We will work with you to understand the issue and coordinate a fix before any public disclosure.

## Scope

As a UI styling library with no network calls, authentication, or data storage, the attack surface is minimal. However, we take all reports seriously — particularly anything related to:

- Dependency vulnerabilities
- Build/supply chain integrity
- Unexpected code execution
